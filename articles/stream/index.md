# Streams

## 模块信息

要使用 Akka Streams，请将模块添加到您的项目中：

```scala
val AkkaVersion = "2.6.15"
libraryDependencies ++= Seq(
  "com.typesafe.akka" %% "akka-stream" % AkkaVersion,
  "com.typesafe.akka" %% "akka-stream-testkit" % AkkaVersion % Test
)
```
